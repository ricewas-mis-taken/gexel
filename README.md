Built with VS Code, Vite, and React, gexel is a browser game built for Radish Jam 2026

Gexel is a browser game that disguises itself as a spreadsheet app. Typing "gexel" takes you to a hidden retro-arcade game collection where you progress to build your "RAM stick" and defeat the evil IT management. Included games include: versions of Pac-Man, Galaga, Frogger, Sega Turbo, Tetris, and Street Fighter 2. Boilerplate code was built with Claude.
